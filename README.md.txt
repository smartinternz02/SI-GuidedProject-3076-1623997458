# SI-GuidedProject-3074-1623997402
Hello, This project is  about Diabetic Mellitus Prediction using IBM AutoAI
This project prevents the people from the avalanche by priory informing them there is a chance to the occurrence of avalanche or not.
In medicine, the diagnosis of diabetes is according to fasting blood glucose, glucose tolerance, and random blood glucose levels. 
The earlier diagnosis is obtained, the much easier we can control it. Machine learning can help people make a preliminary judgment 
about diabetes mellitus according to their daily physical examination data, and it can serve as a reference for doctors.
